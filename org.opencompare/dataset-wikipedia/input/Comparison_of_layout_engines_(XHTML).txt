The following tables compare [[XHTML]] compatibility and support for a number of [[layout engine]]s. Please see the individual products' articles for further information. Unless otherwise specified in footnotes, comparisons are based on the stable versions without any add-ons, extensions or external programs.

Only [[XHTML#XHTML 1.1â€”Module-based XHTML|XHTML 1.1]] is considered. XHTML 1.1 is necessarily [[XHTML#XHTML 1.0|XHTML 1.0]] Strict with more elements and attributes deprecated. [[XHTML#XHTML 2.0|XHTML 2.0]] was a working draft with no layout engine support, but work on it was abandoned<ref>{{cite web | title=W3C XHTML FAQ | url=http://www.w3.org/2009/06/xhtml-faq.html}}</ref> in 2009 in favor of work on [[HTML5]] and [[XHTML#XHTML5|XHTML5]]. XHTML 1.0 and [[HTML|HTML 4]] (both served with <code>text/html</code>) are covered in [[comparison of layout engines (HTML)]]. The comparison of XML capabilities are covered in [[comparison of layout engines (XML)]].

Version number, if possible, is provided for feature that is fully supported. Proprietary extensions are not included.
