The following is a general comparison of [[Glossary of BitTorrent terms#Client|BitTorrent client]]s, which are [[computer program]]s designed for [[peer-to-peer file sharing]] using the [[BitTorrent (protocol)|BitTorrent protocol]].<ref name="btclient">{{cite book |author=<!-- no byline --> |title=BitTorrent User Manual |chapter=Glossary |chapterurl=http://www.bittorrent.com/help/manual/glossary#client |accessdate=2 July 2013 |publisher=[[BitTorrent (company)|BitTorrent, Inc.]] |location=[[San Francisco]]}}</ref>

The BitTorrent protocol coordinates [[segmented file transfer]] among [[Glossary of BitTorrent terms#Peer|peers]] connected in a [[Glossary of BitTorrent terms#Swarm|swarm]]. A BitTorrent client enables a user to exchange data as a peer in one or more swarms. Because BitTorrent is a [[peer-to-peer]] [[communications protocol]] that doesn't need a [[server (computing)|server]], the BitTorrent definition of ''client'' differs from [[Client (computing)|the conventional meaning]] expressed in the [[client–server model]].<ref name="btclient" />

[[Bram Cohen]], author of the BitTorrent protocol, made the first BitTorrent client, which he also called [[BitTorrent (software)|BitTorrent]], and published it in July 2001.<ref>{{cite web |last=Cohen |first=Bram |url=http://finance.groups.yahoo.com/group/decentralization/message/3160|title=BitTorrent - a new P2P app |website=Decentralization · Implications of the end-to-end principle |publisher=[[Yahoo! Groups]] |date=2 July 2001|accessdate=2 July 2013}}</ref>

Many BitTorrent programs are [[open-source software]]; others are [[freeware]], [[adware]] or [[shareware]]. Some [[download manager]]s, such as [[FlashGet]] and [[GetRight]], are BitTorrent-ready. [[Opera (web browser)|Opera 12]], a [[web browser]], can also transfer files via BitTorrent.

In 2007, five BitTorrent clients were discovered to be [[Trojan horse (computing)|Trojan horses]] that attempt to infect [[Microsoft Windows|Windows]] with [[malware]].<ref>{{cite web |url=https://torrentfreak.com/bittorrent-malware-spreads-to-media-players/ |title=BitTorrent Malware Spreads to Media Players |author=<!-- no byline --> |date=22 May 2007 |website=[[TorrentFreak]] |accessdate=3 July 2013}}</ref><ref>{{cite web |url=http://www.leechermods.com/2007/08/malicious-bittorrent-clients-new-coat.html |title=Malicious BitTorrent Clients: New Coat of Paint, Same Bad Story |author=<!-- No byline --> |date=4 August 2007 |website=Leecher Mods |accessdate=3 July 2013}}</ref><ref>{{cite web |url=http://msmvps.com/blogs/hostsnews/archive/2007/07/19/bittorrent-users-beware.aspx |title=BitTorrent users beware! |author=<!-- no byline --> |date=19 July 2007 |website=Hosts News |accessdate=3 July 2013}}</ref> Those clients—BitRoll, GetTorrent, Torrent101, µTorrent,<ref>http://www.trustedreviews.com/news/utorrent-silently-installing-bundled-bitcoin-mining-software?utm_content=buffer433bd&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer</ref> and TorrentQ—are excluded from the comparison tables below. In 2013, Thunder Networking Technologies admitted that some of their employees surreptitiously distributed a Trojan horse with certain releases of [[Xunlei]], the company's BitTorrent-ready download manager.<ref>{{cite web |url=http://torrentfreak.com/google-backed-file-sharing-spreads-malware-131014/ |title=Google-backed BitTorrent client spread malware to Windows PCs and Android devices |last=Van Der Sar |first=Ernesto |date=14 October 2013 |website=[[TorrentFreak]] |accessdate=19 November 2013}}</ref><ref>{{cite web |url=http://www.theguardian.com/technology/2013/oct/15/google-bittorrent-malware-windows-android-xunlei-china |title=Google-Backed File-Sharing Service Spreads Malware |last=Gibbs |first=Samuel |date=15 October 2013 |website=[[The Guardian]] |publisher=Guardian News and Media |accessdate=19 November 2013}}</ref> Xunlei is included in the comparison tables.
